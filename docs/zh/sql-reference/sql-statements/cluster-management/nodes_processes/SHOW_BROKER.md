# SHOW BROKER

SHOW BROKER 用于查看已添加到 StarRocks 集群的所有 Broker 信息。

:::tip

只有拥有 SYSTEM 级 OPERATE 权限或 `cluster_admin` 角色的用户才能执行此操作。

:::

## 语法

```sql
SHOW BROKER
```

## 返回值

```sql
+-----------+-----------+------+-------+---------------------+---------------------+--------+
| Name      | IP        | Port | Alive | LastStartTime       | LastUpdateTime      | ErrMsg |
+-----------+-----------+------+-------+---------------------+---------------------+--------+
```

下表描述了该语句返回的参数：

| **参数**              | **说明**                                                          |
| -------------------- | ----------------------------------------------------------------- |
| Name                 | Broker 的名称。                                                    |
| IP                   | Broker 运行的 IP 地址。                                             |
| Port                 | Broker 使用的端口（默认为 8000）。                                    |
| Alive                | Broker 节点是否可达。 <ul><li>true: Broker 存活。</li><li>false: Broker 不可达。</li></ul> |
| LastStartTime        | Broker 进程最近一次启动的时间。                                       |
| LastUpdateTime       | Broker 向 Frontend (FE) 汇报心跳的最近时间。                          |
| ErrMsg               | 当心跳失败或连接丢失时显示的错误信息。                                   |

## 示例

查看集群中所有 Broker 的信息。

```Plain
SHOW BROKER;

+-----------+-----------+------+-------+---------------------+---------------------+--------+
| Name      | IP        | Port | Alive | LastStartTime       | LastUpdateTime      | ErrMsg |
+-----------+-----------+------+-------+---------------------+---------------------+--------+
| my_broker | 127.0.0.1 | 8000 | true  | 2026-01-21 12:40:00 | 2026-01-21 12:40:05 |        |
+-----------+-----------+------+-------+---------------------+---------------------+--------+
```
