# CANCEL RESTORE

## 功能

该语句用于取消一个正在进行的 [RESTORE](../data-definition/RESTORE.md) 任务。

## 语法

```sql
CANCEL RESTORE FROM db_name;
```

注意：
当取消处于 `COMMIT` 或之后阶段的恢复任务时，可能导致被恢复的表无法访问。此时只能通过再次执行恢复作业进行数据恢复。

## 示例

1. 取消 example_db 下的 RESTORE 任务。

    ```sql
    CANCEL RESTORE FROM example_db;
    ```

## 关键字(keywords)

CANCEL, RESTORE
