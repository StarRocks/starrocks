# inspect_mv_meta

`inspect_mv_meta(mv_name)`

此函数返回物化视图的详细元数据。

## 参数

`mv_name`: 物化视图的名称 (VARCHAR)。

## 返回值

返回包含物化视图详细元数据的 VARCHAR 字符串。
