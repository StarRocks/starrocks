# inspect_hive_part_info

`inspect_hive_part_info(table_name)`

此函数返回表的 Hive 分区信息。

## 参数

`table_name`: 表的名称 (VARCHAR)。

## 返回值

返回包含 Hive 分区信息的 JSON 格式的 VARCHAR 字符串。

## 示例

示例 1: 检查分区 Hive 表的分区信息:
```
mysql> select inspect_hive_part_info('t1');
```
