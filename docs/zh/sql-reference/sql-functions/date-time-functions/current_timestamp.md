# current_timestamp

## 功能

获取当前时间，以 DATETIME 类型返回。

## 语法

```Haskell
DATETIME CURRENT_TIMESTAMP()
```

## 示例

```Plain Text
select current_timestamp();
+---------------------+
| current_timestamp() |
+---------------------+
| 2019-05-27 15:59:33 |
+---------------------+
```
