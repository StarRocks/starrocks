# quarters_add



向日期添加指定的季度数。一个季度等于三个月。

## 语法

```Haskell
DATETIME quarters_add(DATETIME expr1, INT expr2);
```

## 参数说明

`expr1`: 待添加季度数的时间，支持的数据类型为 DATETIME 或 DATE。

`expr2`: 增加的季度数，支持的数据类型为 INT。取值可以大于，等于或者小于 0。负数表示从 `expr1` 中减去相应的季度数。

## 返回值说明

返回值的数据类型为 DATETIME。

## 示例

```Plain
select quarters_add('2019-08-01 13:21:03', 1);
+----------------------------------------+
| quarters_add('2019-08-01 13:21:03', 1) |
+----------------------------------------+
| 2019-11-01 13:21:03                    |
+----------------------------------------+

select quarters_add('2019-08-01', 1);
+-------------------------------+
| quarters_add('2019-08-01', 1) |
+-------------------------------+
| 2019-11-01 00:00:00           |
+-------------------------------+

select quarters_add('2019-08-01 13:21:03', -1);
+-----------------------------------------+
| quarters_add('2019-08-01 13:21:03', -1) |
+-----------------------------------------+
| 2019-05-01 13:21:03                     |
+-----------------------------------------+
```
