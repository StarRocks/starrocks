# quarters_sub



从日期中减去指定的季度数。一个季度等于三个月。

## 语法

```Haskell
DATETIME quarters_sub(DATETIME expr1, INT expr2);
```

## 参数说明

`expr1`: 指定的日期，支持的数据类型为 DATETIME 或 DATE。

`expr2`: 被减去的季度数，支持的数据类型为 INT。取值可以大于，等于或者小于 0。负数表示向 `expr1` 中添加相应的季度数。

## 返回值说明

返回值的数据类型为 DATETIME。

## 示例

```Plain
select quarters_sub('2022-02-28 15:04:05', 1);
+----------------------------------------+
| quarters_sub('2022-02-28 15:04:05', 1) |
+----------------------------------------+
| 2021-11-28 15:04:05                    |
+----------------------------------------+

select quarters_sub('2022-02-28', 1);
+-------------------------------+
| quarters_sub('2022-02-28', 1) |
+-------------------------------+
| 2021-11-28 00:00:00           |
+-------------------------------+
```
