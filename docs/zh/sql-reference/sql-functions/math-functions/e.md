# e

## 功能

返回自然对数函数的底数。

## 语法

```Haskell
E();
```

## 参数说明

无

## 返回值说明

返回值的数据类型为 DOUBLE。

## 示例

```Plain Text
mysql> select e();
+-------------------+
| e()               |
+-------------------+
| 2.718281828459045 |
+-------------------+
1 row in set (0.01 sec)
```
