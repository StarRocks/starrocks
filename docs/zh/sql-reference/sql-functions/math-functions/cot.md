# cot

## 功能

求参数 `x` 的余切值，`x` 是弧度

## 语法

```Haskell
COT(x);
```

## 参数说明

`x`: 支持的数据类型为 DOUBLE

## 返回值说明

返回值的数据类型为 DOUBLE

## 示例

```Plain Text
mysql> select cot(6);
+--------------------+
| cot(6)             |
+--------------------+
| -3.436353004180128 |
+--------------------+
1 row in set (0.00 sec)
```
