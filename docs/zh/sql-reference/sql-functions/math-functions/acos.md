# acos

## 功能

求 `x` 的反余弦值(单位为弧度)，x 为一个数值

## 语法

```Haskell
ACOS(x);
```

## 参数说明

`x`: 支持的数据类型为 DOUBLE

## 返回值说明

返回值的数据类型为 DOUBLE

## 示例

```Plain Text
mysql> select acos(0.25);
+-------------------+
| acos(0.25)        |
+-------------------+
| 1.318116071652818 |
+-------------------+
1 row in set (0.02 sec)
```
