# pi

## 功能

返回圆周率。

## 语法

```Haskell
PI();
```

## 参数说明

无

## 返回值说明

返回值的数据类型为 DOUBLE。

## 示例

```Plain Text
mysql> select pi();
+-------------------+
| pi()              |
+-------------------+
| 3.141592653589793 |
+-------------------+
1 row in set (0.00 sec)
```
