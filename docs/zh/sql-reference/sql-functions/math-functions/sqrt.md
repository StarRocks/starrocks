# sqrt, dsqrt

## 功能

返回参数 `x` 的平方根

## 语法

```Haskell
SQRT(x);
```

## 参数说明

`x`: 支持的数据类型为 DOUBLE

## 返回值说明

返回值的数据类型为 DOUBLE

## 示例

```Plain Text
mysql> select sqrt(3.14);
+-------------------+
| sqrt(3.14)        |
+-------------------+
| 1.772004514666935 |
+-------------------+
1 row in set (0.01 sec)
```

## 关键词

SQRT, DSQRT
