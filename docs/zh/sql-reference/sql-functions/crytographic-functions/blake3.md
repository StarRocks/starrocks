# blake3

使用 BLAKE3 摘要算法将字符串哈希为一个 256 位的十六进制字符串。每 32 位之间用空格分隔。

BLAKE3 是一种现代加密哈希函数，其速度显著快于 MD5、SHA-1、SHA-2 和 SHA-3，同时提供强大的安全保证。它生成一个 256 位（32 字节）的摘要。

## 语法

```Haskell
BLAKE3(str);
```

## 参数说明

`str`: 待哈希的字符串，支持的数据类型为 VARCHAR。

## 返回值说明

返回值的数据类型为 VARCHAR。如果输入为 NULL，返回结果为 NULL。空字符串输入返回长度为零的消息的 BLAKE3 摘要 (`af1349b9 f5f9a1a6 a0404dea 36dcc949 9bcb25c9 adc112b7 cc9a93ca e41f3262`)。

该函数只接受一个字符串。如果输入多个字符串，则会报错。

## 示例

```Plain Text
mysql> select blake3('l');
+-------------------------------------------------------------------------+
| blake3('l')                                                             |
+-------------------------------------------------------------------------+
| b9f63c83 975dadee 9b4dc3fa eeb669dd ef34bbe1 2e3a0626 18316698 2681d81c |
+-------------------------------------------------------------------------+
1 row in set (0.01 sec)
```
