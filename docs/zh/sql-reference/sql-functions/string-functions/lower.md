# lower

## description

### Syntax

```Haskell
INT lower(VARCHAR str)
```

将参数中所有的字符串都转换成小写

## example

```Plain Text
MySQL > SELECT lower("AbC123");
+-----------------+
| lower('AbC123') |
+-----------------+
| abc123          |
+-----------------+
```

## keyword

LOWER
