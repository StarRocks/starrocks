# space

## 功能

返回参数 `x` 指定的 `x` 个空格。

## 语法

```Haskell
space(x);
```

## 参数说明

`x`: 支持的数据类型为 INT。

## 返回值说明

返回值的数据类型为 VARCHAR。

## 示例

```Plain Text
mysql> select space(6);
+----------+
| space(6) |
+----------+
|          |
+----------+
1 row in set (0.00 sec)
```
