# be_metrics

`be_metrics` 提供有关每个 BE 节点指标的信息。

`be_metrics` 提供以下字段：

| **字段** | **描述**                                         |
| -------- | ------------------------------------------------ |
| BE_ID    | BE 节点的 ID。                                   |
| NAME     | 指标的名称。                                     |
| LABELS   | 与指标关联的标签。                               |
| VALUE    | 指标的当前值。                                   |
