# be_configs

`be_configs` 提供有关每个 BE 节点的配置参数信息。

`be_configs` 提供以下字段：

| **字段** | **描述**                                         |
| -------- | ------------------------------------------------ |
| BE_ID    | BE 节点的 ID。                                   |
| NAME     | 配置参数的名称。                                 |
| VALUE    | 配置参数的当前值。                               |
| TYPE     | 配置参数的数据类型。                             |
| DEFAULT  | 配置参数的默认值。                               |
| MUTABLE  | 指示配置参数是否可以动态修改（`true`）或不能（`false`）。 |
