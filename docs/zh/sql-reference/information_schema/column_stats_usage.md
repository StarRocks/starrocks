# column_stats_usage

`column_stats_usage` 提供有关列统计信息使用情况的信息。

`column_stats_usage` 提供以下字段：

| **字段**       | **描述**                                         |
| -------------- | ------------------------------------------------ |
| TABLE_CATALOG  | 表的 Catalog 名称。                              |
| TABLE_DATABASE | 表的数据库名称。                                 |
| TABLE_NAME     | 表的名称。                                       |
| COLUMN_NAME    | 列的名称。                                       |
| USAGE          | 此列统计信息的使用场景。                         |
| LAST_USED      | 上次使用此列统计信息的时间。                     |
| CREATED        | 此列统计信息的创建时间。                         |
