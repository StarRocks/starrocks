# be_logs

`be_logs` 提供有关每个 BE 节点日志的信息。

`be_logs` 提供以下字段：

| **字段**    | **描述**                                         |
| ----------- | ------------------------------------------------ |
| BE_ID       | BE 节点的 ID。                                   |
| LEVEL       | 日志级别（例如，`INFO`、`WARNING`、`ERROR`）。      |
| TIMESTAMP   | 日志条目的时间戳。                               |
| TID         | 生成日志条目的线程 ID。                          |
| LOG         | 日志消息内容。                                   |
