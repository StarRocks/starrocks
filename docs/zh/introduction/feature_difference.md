# 无共享集群和共享数据集群之间的功能差异

本主题列出了无共享集群和共享数据集群支持的功能差异。

## 实时分析和主键表

| 功能 | 无共享 | 共享数据 |
| ---------------------------------------------------- | -------------- | --------------- |
| 条件更新 | v2.5+ | v3.1+ |
| 列模式部分更新 | v3.1 | 待支持 |
| 行模式部分更新 | v2.3+ | v3.1+ |
| 带条件更新的部分更新 | v3.1+ | v3.4.1+ |
| 主键索引持久化 | v2.3+ | v3.2+ |
| 主键与 ORDER BY 列解耦 | v3.0+ | v3.1+ |
| 主键表的丰富 UPDATE 和 DELETE 语法 | v3.0+ | v3.1+ |

## 存储引擎

| 功能 | 无共享 | 共享数据 |
| ---------------------------------------------------- | -------------- | --------------- |
| 快速 Schema 演进 | v3.2+ | v4.0+ |
| 文件捆绑 | 不适用 | v4.0+ |
| 倒排索引 | v3.3+ | v4.1+ |
| 手动 Compaction | v3.1+ | v3.3+ |
| 行存储 | v3.2+ | 待支持 |
| 添加/删除 STRUCT 字段 | v3.3.2+ | v3.3.5+ |
| 向量索引 | v3.4+ | 待支持 |

## 数据分布

| 功能 | 无共享 | 共享数据 |
| ---------------------------------------------------- | -------------- | --------------- |
| 表达式分区 | v3.0+ | v3.1.1+ |
| 列表分区 | v3.1+ | v3.1.1+ |
| 随机分桶 | v3.1+ | v3.2+ |
| 随机分桶优化（带子分区） | v3.2+ | v3.2+ |
| 多级表达式分区的统一语法 | v3.4+ | v3.4+ |
| 优化表：更改分桶 | v3.2+ | v3.3+ |
| 优化表：在线更改分桶 | v3.3.3+ | 不适用 |
| 基于范围的分布 | v4.1+ | v4.1+ |

## 查询性能

| 功能 | 无共享 | 共享数据 |
| ---------------------------------------------------- | -------------- | --------------- |
| 跨节点数据缓存共享 | 不适用 | v3.5.1+ |
| 生成列 | v3.1+ | v3.5+ |
| 查询缓存 | v2.5+ | v3.4+ |
| Rollup 和同步物化视图 | v1.x+ | v3.3+ |
