# tasks

`tasks` 提供有关异步任务的信息。

`tasks` 提供以下字段：

| **字段**    | **描述**                                                     |
| ----------- | ------------------------------------------------------------ |
| TASK_NAME   | 任务的名称。                                                 |
| CREATE_TIME | 任务创建的时间。                                             |
| SCHEDULE    | 任务调度。如果任务为周期运行，会显示 `START xxx EVERY xxx`。 |
| DATABASE    | 任务所属的数据库。                                           |
| DEFINITION  | 任务的 SQL 定义。                                            |
| EXPIRE_TIME | 任务过期的时间。                                             |
