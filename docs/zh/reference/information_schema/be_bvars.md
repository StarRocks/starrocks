# be_bvars

`be_bvars` 提供有关 bRPC 的统计信息。您可以查看 RPC 延迟、QPS 和某些 StarRocks 组件的统计信息。

`be_bvars` 提供以下字段：

| **字段** | **描述**                          |
| -------- | --------------------------------- |
| BE_ID    | bvar 所在的 BE 的 ID。             |
| NAME     | bvar 名称。                       |
| DESC     | bvar 描述，其中包括统计信息的值。 |
