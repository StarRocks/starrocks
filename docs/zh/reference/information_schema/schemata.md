# schemata

`schemata` 提供有关数据库的信息。

`schemata` 提供以下字段：

| 字段                       | 描述                                    |
| -------------------------- | --------------------------------------- |
| CATALOG_NAME               | 模式所属的目录的名称。该值始终为 NULL。 |
| SCHEMA_NAME                | 模式的名称。                            |
| DEFAULT_CHARACTER_SET_NAME | 模式的默认字符集。                      |
| DEFAULT_COLLATION_NAME     | 模式的默认排序规则。                    |
| SQL_PATH                   | 此值始终为 NULL。                       |
