# collations

`collations` 包含可用的排序规则。

`collations` 提供以下字段：

| **字段**           | **描述**                                       |
| ------------------ | ---------------------------------------------- |
| COLLATION_NAME     | 排序规则名称。                                 |
| CHARACTER_SET_NAME | 与排序规则相关联的字符集的名称。               |
| ID                 | 排序规则的 ID。                                |
| IS_DEFAULT         | 排序规则是否为其字符集的默认值。               |
| IS_COMPILED        | 字符集是否已编译到服务器中。                   |
| SORTLEN            | 与在字符集中表达的字符串排序所需的内存量有关。 |
