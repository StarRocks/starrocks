## Key 列和 Value 列的数据类型

表中 Key 列支持数据类型为数值（包括整型、布尔和 DECIMAL）、字符串、时间日期。

:::note

主键表的 Key 列不支持为 DECIMAL。

:::

而表中 Value 列支持基础的数据类型，包括数值、字符串、时间日期。不同类型的表中 Value 列对于 BITMAP、HLL 以及半结构化类型的支持度不同，具体如下：
