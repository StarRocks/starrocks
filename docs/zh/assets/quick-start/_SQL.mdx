#### 能见度不佳的情况下在纽约市驾驶是否安全？

为了回答这个问题，需要查看在能见度较差（0 到 1.0 英里之间）时的交通事故数量。在两个表格上对 DATETIME 列进行 JOIN。

```sql
SELECT COUNT(DISTINCT c.COLLISION_ID) AS Crashes,
       truncate(avg(w.HourlyDryBulbTemperature), 1) AS Temp_F,
       truncate(avg(w.HourlyVisibility), 2) AS Visibility,
       max(w.HourlyPrecipitation) AS Precipitation,
       date_format((date_trunc("hour", c.CRASH_DATE)), '%d %b %Y %H:%i') AS Hour
FROM crashdata c
LEFT JOIN weatherdata w
ON date_trunc("hour", c.CRASH_DATE)=date_trunc("hour", w.DATE)
WHERE w.HourlyVisibility BETWEEN 0.0 AND 1.0
GROUP BY Hour
ORDER BY Crashes DESC
LIMIT 100;
```

从以上可以得出，在能见度较低的一个小时内的最高交通事故数量是 129。有多个因素需要考虑在内：

- 2014 年 2 月 3 日是星期一
- 上午 8 点是早高峰时段
- 当时正在下雨（那个小时的降水量为 0.12 英寸）
- 温度为 32 华氏度（水的冰点）
- 能见度为 0.25 英里，纽约市的正常能见度为 10 英里

#### 在结冰条件下驾驶是否安全？

水汽在 40 华氏度时会转变为冰；因此以下查询关注温度为 0 到 40 华氏度的区间。

```sql
SELECT COUNT(DISTINCT c.COLLISION_ID) AS Crashes,
       truncate(avg(w.HourlyDryBulbTemperature), 1) AS Temp_F,
       truncate(avg(w.HourlyVisibility), 2) AS Visibility,
       max(w.HourlyPrecipitation) AS Precipitation,
       date_format((date_trunc("hour", c.CRASH_DATE)), '%d %b %Y %H:%i') AS Hour
FROM crashdata c
LEFT JOIN weatherdata w
ON date_trunc("hour", c.CRASH_DATE)=date_trunc("hour", w.DATE)
WHERE w.HourlyDryBulbTemperature BETWEEN 0.0 AND 40.5 
GROUP BY Hour
ORDER BY Crashes DESC
LIMIT 100;
```

结果显示在寒冷的一月，星期天早晨城市中仍有大量交通事故。根据 [weather.com](https://weather.com/storms/winter/news/northeast-storm-rain-snow-wind) 显示，当天有一场大风暴，导致许多交通事故，这正如查询结果中所显示的一样。
