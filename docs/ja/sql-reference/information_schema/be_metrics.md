# be_metrics

`be_metrics` は各 BE ノードのメトリックに関する情報を提供します。

`be_metrics` には以下のフィールドが提供されています:

| **フィールド** | **説明**                                         |
| -------- | ------------------------------------------------ |
| BE_ID    | BE ノードの ID。                                   |
| NAME     | メトリックの名前。                               |
| LABELS   | メトリックに関連付けられたラベル。               |
| VALUE    | メトリックの現在の値。                           |
