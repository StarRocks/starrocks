# session_variables

`session_variables` はセッション変数に関する情報を提供します。

`session_variables` には以下のフィールドが含まれています:

| **Field**      | **Description**        |
| -------------- | ---------------------- |
| VARIABLE_NAME  | 変数の名前。           |
| VARIABLE_VALUE | 変数の値。             |
