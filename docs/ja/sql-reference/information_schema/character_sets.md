# character_sets

`character_sets` は利用可能な文字セットを識別します。

`character_sets` には次のフィールドが提供されています:

| **Field**            | **Description**                                      |
| -------------------- | ---------------------------------------------------- |
| CHARACTER_SET_NAME   | 文字セットの名前。                                   |
| DEFAULT_COLLATE_NAME | 文字セットのデフォルトの照合順序。                   |
| DESCRIPTION          | 文字セットの説明。                                   |
| MAXLEN               | 1文字を格納するのに必要な最大バイト数。               |
