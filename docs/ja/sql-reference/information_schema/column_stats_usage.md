# column_stats_usage

`column_stats_usage` は列統計情報の使用状況に関する情報を提供します。

`column_stats_usage` には以下のフィールドが提供されています:

| **フィールド**       | **説明**                                         |
| -------------- | ------------------------------------------------ |
| TABLE_CATALOG  | テーブルのカタログ名。                           |
| TABLE_DATABASE | テーブルのデータベース名。                       |
| TABLE_NAME     | テーブル名。                                     |
| COLUMN_NAME    | 列名。                                           |
| USAGE          | この列統計情報の使用例。                         |
| LAST_USED      | この列統計情報が最後に使用された時刻。           |
| CREATED        | この列統計情報の作成時刻。                       |
