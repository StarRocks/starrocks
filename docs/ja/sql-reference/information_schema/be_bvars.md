# be_bvars

`be_bvars` は bRPC に関する統計情報を提供します。StarRocks の一部コンポーネントの RPC レイテンシー、QPS、その他の統計情報を表示できます。

`be_bvars` には以下のフィールドが提供されています:

| **フィールド** | **説明**                                         |
| --------- | ------------------------------------------------ |
| BE_ID     | bvar が配置されている BE の ID。                   |
| NAME      | bvar の名前。                                    |
| VALUE     | bvar の値。                                      |
