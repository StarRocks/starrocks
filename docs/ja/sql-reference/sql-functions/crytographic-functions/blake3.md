# blake3

BLAKE3 ダイジェストアルゴリズムを使用して、文字列を 256 ビットの 16 進文字列にハッシュします。

BLAKE3 は、MD5、SHA-1、SHA-2、SHA-3 よりも大幅に高速でありながら、強力なセキュリティ保証を提供する最新の暗号学的ハッシュ関数です。256 ビット (32 バイト) のダイジェストを生成します。

## 構文

```Haskell
BLAKE3(str);
```

## パラメータ

`str`: ハッシュする文字列。VARCHAR 型である必要があります。

## 戻り値

VARCHAR 型の値を返します。入力が NULL の場合、NULL が返されます。空の入力文字列は、長さ 0 のメッセージの BLAKE3 ダイジェスト (`af1349b9f5f9a1a6a0404dea36dcc9499bcb25c9adc112b7cc9a93cae41f3262`) を返します。

この関数は 1 つの文字列のみを受け入れます。複数の入力文字列を指定するとエラーになります。

## 例

```Plain Text
mysql> select blake3('l');
+------------------------------------------------------------------+
| blake3('l')                                                      |
+------------------------------------------------------------------+
| b9f63c83975dadee9b4dc3faeeb669ddef34bbe12e3a0626183166982681d81c |
+------------------------------------------------------------------+
1 row in set (0.01 sec)
```
