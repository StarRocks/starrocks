# inspect_mv_meta

`inspect_mv_meta(mv_name)`

この関数は、マテリアライズドビューの詳細なメタデータを返します。

## 引数

`mv_name`: マテリアライズドビューの名前 (VARCHAR)。

## 戻り値

マテリアライズドビューの詳細なメタデータを含む VARCHAR 文字列を返します。
