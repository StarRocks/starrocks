# degrees

ラジアン `x` を度に変換します。

## 構文

```SQL
DEGREES(x);
```

## パラメータ

`x`: ラジアンでの角度。DOUBLE がサポートされています。

## 戻り値

DOUBLE データ型の値を返します。

## 例

```Plaintext
mysql> select degrees(3.1415926535898);
+--------------------------+
| degrees(3.1415926535898) |
+--------------------------+
|        180.0000000000004 |
+--------------------------+
1 row in set (0.07 sec)
```
