# asin

引数のアークサインを計算します。

## 構文

```Haskell
DOUBLE asin(DOUBLE arg)
```

### パラメータ

`arg`: 数値のみを指定できます。この関数は、数値をアークサインを計算する前に DOUBLE 値に変換します。

## 戻り値

DOUBLE データ型の値を返します。

## 使用上の注意

非数値を指定した場合、この関数は `NULL` を返します。

## 例

```Plain
mysql> select asin(1);
+--------------------+
| asin(1)            |
+--------------------+
| 1.5707963267948966 |
+--------------------+

mysql> select asin(-0.5);
+---------------------+
| asin(-0.5)          |
+---------------------+
| -0.5235987755982989 |
+---------------------+

mysql> select asin(0);
+---------+
| asin(0) |
+---------+
|       0 |
+---------+

mysql> select asin("");
+----------+
| asin('') |
+----------+
|     NULL |
+----------+
```
