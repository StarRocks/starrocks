# rand, random

## 説明

0（含む）から1（含まない）までのランダムな浮動小数点数を返します。

## 構文

```Haskell
RAND(x);
```

## パラメータ

`x`: オプションです。データ型は BIGINT です。`x` が指定されているかどうかに関わらず、この関数は完全にランダムな数を返します。

## 戻り値

DOUBLE 型の値を返します。

## 例

```Plain Text
select rand();
+--------------------+
| rand()             |
+--------------------+
| 0.9393535880089522 |
+--------------------+
1 row in set (0.01 sec)
select rand(3);
+--------------------+
| rand(3)            |
+--------------------+
| 0.6659865964511347 |
+--------------------+
1 row in set (0.00 sec)
```

## キーワード

RAND, RANDOM
