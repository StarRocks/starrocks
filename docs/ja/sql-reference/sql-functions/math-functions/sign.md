# sign

## 説明

`x` の符号を返します。入力が負の数、0、または正の数の場合、それぞれ `-1`、`0`、または `1` を出力します。

## 構文

```Haskell
SIGN(x);
```

## パラメータ

`x`: DOUBLE データ型をサポートします。

## 戻り値

FLOAT データ型の値を返します。

## 例

```Plain
mysql> select sign(3.14159);
+---------------+
| sign(3.14159) |
+---------------+
|             1 |
+---------------+
1 row in set (0.02 sec)
```
