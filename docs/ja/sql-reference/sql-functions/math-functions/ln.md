# ln, dlog1, log

数値の自然対数（底 `e`）を計算します。

## 構文

```SQL
ln(arg)
log(arg)
dlog1(arg)
```

## パラメータ

- `arg`: 対数を計算したい値。DOUBLE データ型のみサポートされています。

> **注意**
>
> `arg` が負または 0 に指定された場合、StarRocks は NULL を返します。

## 戻り値

DOUBLE データ型の値を返します。

## 例

例 1: 3 の自然対数を計算します。

```SQL
mysql> select ln(3);
+--------------------+
| ln(3)              |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)

mysql> select dlog1(3);
+--------------------+
| dlog1(3)              |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)

mysql> select log(3);
+--------------------+
| log(3)              |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)

```
