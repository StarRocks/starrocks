# tan

## 説明

`x` の正接を返します。ここで、`x` はラジアンで指定されます。

## 構文

```Haskell
TAN(x);
```

## パラメータ

`x`: DOUBLE データ型をサポートします。

## 戻り値

DOUBLE データ型の値を返します。

## 例

```Plain
mysql> select tan(3.14);
+-----------------------+
| tan(3.14)             |
+-----------------------+
| -0.001592654936407223 |
+-----------------------+
1 row in set (0.12 sec)
```
