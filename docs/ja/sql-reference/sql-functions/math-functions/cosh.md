# cosh

引数の双曲線余弦を計算します。

この関数は v3.0 からサポートされています。

## 構文

```Haskell
DOUBLE cosh(DOUBLE arg)
```

### パラメータ

`arg`: 数値のみ指定できます。この関数は、数値を DOUBLE 値に変換してから、その値の双曲線余弦を計算します。

## 戻り値

DOUBLE データ型の値を返します。

## 使用上の注意

非数値を指定した場合、この関数は `NULL` を返します。

## 例

```Plain
mysql> select cosh(-1);
+--------------------+
| cosh(-1)           |
+--------------------+
| 1.5430806348152437 |
+--------------------+

mysql> select cosh(0);
+---------+
| cosh(0) |
+---------+
|       1 |
+---------+

mysql> select cosh(1);
+--------------------+
| cosh(1)            |
+--------------------+
| 1.5430806348152437 |
+--------------------+

mysql> select cosh("");
+----------+
| cosh('') |
+----------+
|     NULL |
+----------+
```
