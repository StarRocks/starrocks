# sin

## 説明

`x` のサインを返します。ここで `x` はラジアンです。

## 構文

```Haskell
SIN(x);
```

## パラメータ

`x`: DOUBLE データ型をサポートします。

## 戻り値

DOUBLE データ型の値を返します。

## 例

```Plain
mysql> select sin(3.14);
+-----------------------+
| sin(3.14)             |
+-----------------------+
| 0.0015926529164868282 |
+-----------------------+
1 row in set (0.21 sec)
```
