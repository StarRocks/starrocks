# log10, dlog10

## 説明

数値の常用対数（底が10の対数）を計算します。

## 構文

```SQL
log10(arg)
```

## パラメータ

- `arg`: 対数を計算したい値。DOUBLE データ型のみサポートされています。

> **注意**
>
> StarRocks は、`arg` が負の値または0の場合、NULLを返します。

## 戻り値

DOUBLE データ型の値を返します。

## 例

例 1: 100の常用対数を計算します。

```Plain
select log10(100);
+------------+
| log10(100) |
+------------+
|          2 |
+------------+
1 row in set (0.02 sec)
```
