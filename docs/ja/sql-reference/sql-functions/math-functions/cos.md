# cos

## 説明

引数のコサインを計算します。

## 構文

```Haskell
DOUBLE cos(DOUBLE arg)
```

### パラメータ

`arg`: 数値のみを指定できます。この関数は、数値をコサイン計算の前に DOUBLE 値に変換します。

## 戻り値

DOUBLE データ型の値を返します。

## 使用上の注意

非数値を指定した場合、この関数は `NULL` を返します。

## 例

```Plain
mysql> select cos(-1);
+--------------------+
| cos(-1)            |
+--------------------+
| 0.5403023058681398 |
+--------------------+

mysql> select cos(0);
+--------+
| cos(0) |
+--------+
|      1 |
+--------+

mysql> select cos(1);
+--------------------+
| cos(1)             |
+--------------------+
| 0.5403023058681398 |
+--------------------+

mysql> select cos("");
+---------+
| cos('') |
+---------+
|    NULL |
+---------+
```

## キーワード

COS
