# quarters_sub

指定された四半期数を日付から減算します。1 四半期は 3 か月に相当します。

## Syntax

```Haskell
DATETIME quarters_sub(DATETIME expr1, INT expr2);
```

## Parameters

- `expr1`: 指定された日付。DATETIME または DATE 型でなければなりません。

- `expr2`: 減算する四半期数。INT 型でなければなりません。0 より大きい、0 に等しい、または 0 より小さい値を取ることができます。負の値は `expr1` に四半期を加算します。

## Return value

DATETIME 値を返します。

## Examples

```Plain
select quarters_sub('2022-02-28 15:04:05', 1);
+----------------------------------------+
| quarters_sub('2022-02-28 15:04:05', 1) |
+----------------------------------------+
| 2021-11-28 15:04:05                    |
+----------------------------------------+

select quarters_sub('2022-02-28', 1);
+-------------------------------+
| quarters_sub('2022-02-28', 1) |
+-------------------------------+
| 2021-11-28 00:00:00           |
+-------------------------------+
```
