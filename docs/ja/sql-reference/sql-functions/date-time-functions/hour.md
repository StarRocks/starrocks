# hour

## 説明

指定された日付の時間を返します。戻り値は0から23の範囲です。

`date` パラメータは DATE または DATETIME 型でなければなりません。

## 構文

```Haskell
INT HOUR(DATETIME|DATE date)
```

## 例

```Plain Text
MySQL > select hour('2018-12-31 23:59:59');
+-----------------------------+
| hour('2018-12-31 23:59:59') |
+-----------------------------+
|                          23 |
+-----------------------------+
```

## キーワード

HOUR
