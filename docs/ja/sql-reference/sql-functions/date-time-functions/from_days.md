# from_days

0000-01-01 からの日付を返します。

## Syntax

```Haskell
DATE FROM_DAYS(INT N)
```

## Examples

```Plain Text
MySQL > select from_days(730669);
+-------------------+
| from_days(730669) |
+-------------------+
| 2000-07-03        |
+-------------------+
```

## keyword

FROM_DAYS,FROM,DAYS
