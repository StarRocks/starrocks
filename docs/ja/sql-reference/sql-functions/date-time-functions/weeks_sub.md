# weeks_sub

## 説明

指定された週数を日付または日時の値から減算します。

## 構文

```Haskell
DATETIME weeks_sub(DATETIME expr1, INT expr2);
```

## パラメータ

- `expr1`: 元の日付。`DATETIME` 型である必要があります。

- `expr2`: 週の数。`INT` 型である必要があります。

## 戻り値

`DATETIME` 値を返します。

日付が存在しない場合は `NULL` を返します。

## 例

```Plain
select weeks_sub('2022-12-22',2);
+----------------------------+
| weeks_sub('2022-12-22', 2) |
+----------------------------+
|        2022-12-08 00:00:00 |
+----------------------------+
```
