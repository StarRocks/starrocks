# current_timestamp

現在のタイムゾーンを取得し、VARCHAR 値を返します。

## Syntax

```Haskell
VARCHAR CURRENT_TIMEZONE()
```

## Examples

```Plain Text
MySQL > select current_timezone();
+---------------------+
| current_timezone()  |
+---------------------+
| America/Los_Angeles |
+---------------------+
```
