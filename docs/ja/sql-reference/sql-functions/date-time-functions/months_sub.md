# months_sub

指定された月を日付から減算します。

## 構文

```Haskell
DATETIME months_sub(DATETIME expr1, INT expr2);
```

## パラメータ

- `expr1`: 指定された日付。DATETIME または DATE 型でなければなりません。

- `expr2`: 減算する月。INT 型でなければなりません。

## 戻り値

DATETIME 値を返します。

## 例

```Plain Text
select months_sub('2022-02-28 15:04:05', 1);
+--------------------------------------+
| months_sub('2022-02-28 15:04:05', 1) |
+--------------------------------------+
| 2022-01-28 15:04:05                  |
+--------------------------------------+

select months_sub('2022-02-28', 1);
+-----------------------------+
| months_sub('2022-02-28', 1) |
+-----------------------------+
| 2022-01-28 00:00:00         |
+-----------------------------+
```
