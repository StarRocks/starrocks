# DROP RESOURCE GROUP

## 説明

指定されたリソースグループを削除します。

## 構文

```SQL
DROP RESOURCE GROUP <resource_group_name>
```

## パラメータ

| **パラメータ**      | **説明**                                   |
| ------------------- | ----------------------------------------- |
| resource_group_name | 削除するリソースグループの名前です。       |

## 例

例 1: リソースグループ `rg1` を削除します。

```SQL
DROP RESOURCE GROUP rg1;
```
