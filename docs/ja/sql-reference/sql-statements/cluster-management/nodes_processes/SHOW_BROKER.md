# SHOW BROKER

SHOW BROKER は、StarRocks クラスターに追加されたすべてのブローカーの情報を表示します。

:::tip

この操作を実行できるのは、SYSTEM レベルの OPERATE 権限を持つユーザーまたは `cluster_admin` ロールを持つユーザーのみです。

:::

## 構文

```sql
SHOW BROKER
```

## 戻り値

```sql
+-----------+-----------+------+-------+---------------------+---------------------+--------+
| Name      | IP        | Port | Alive | LastStartTime       | LastUpdateTime      | ErrMsg |
+-----------+-----------+------+-------+---------------------+---------------------+--------+
```

以下の表は、このステートメントによって返されるパラメーターについて説明しています。

| **パラメーター**       | **説明**                                                           |
| -------------------- | ----------------------------------------------------------------- |
| Name                 | ブローカーの名前。                                                   |
| IP                   | ブローカーが実行されているIPアドレス。                                  |
| Port                 | ブローカーが使用するポート（デフォルト: 8000）。                          |
| Alive                | ブローカーノードが到達可能かどうか。 <ul><li>true: ブローカーは生存しています。</li><li>false: ブローカーは到達不能です。</li></ul> |
| LastStartTime        | ブローカープロセスが最後に開始された時間。                               |
| LastUpdateTime       | ブローカーがフロントエンド (FE) にハートビートを報告した最後の時間。        |
| ErrMsg               | ハートビートが失敗した場合や接続が失われた場合にエラーメッセージを表示します。 |

## 例

クラスター内のすべてのブローカーの情報を表示します。

```Plain
SHOW BROKER;

+-----------+-----------+------+-------+---------------------+---------------------+--------+
| Name      | IP        | Port | Alive | LastStartTime       | LastUpdateTime      | ErrMsg |
+-----------+-----------+------+-------+---------------------+---------------------+--------+
| my_broker | 127.0.0.1 | 8000 | true  | 2026-01-21 12:40:00 | 2026-01-21 12:40:05 |        |
+-----------+-----------+------+-------+---------------------+---------------------+--------+
```
