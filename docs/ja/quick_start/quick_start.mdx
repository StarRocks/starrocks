# クイックスタート

このクイックスタートガイドは、小規模な StarRocks 環境での作業を始めるのに役立ちます。起動するクラスターは、StarRocks の動作を学ぶのに適していますが、大規模なデータセットの分析やパフォーマンステストには適していません。

スケーラブルなデプロイメントについては、[StarRocks のデプロイ](../deployment/deployment_overview.md)をご覧ください。
