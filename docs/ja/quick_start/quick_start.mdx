# クイックスタート

これらのクイックスタートガイドは、小規模な StarRocks 環境での作業を開始するのに役立ちます。起動するクラスターは、StarRocks の動作を学ぶのに適していますが、大規模なデータセットの分析やパフォーマンステストには適していません。

スケーラブルなデプロイメントについては、 [deploying StarRocks](../deployment/deployment_overview.md) を参照してください。
