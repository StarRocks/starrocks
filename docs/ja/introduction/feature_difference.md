# Shared-nothing と Shared-data クラスター間の機能の違い

このトピックでは、Shared-nothing クラスターと Shared-data クラスターでサポートされる機能の違いを一覧にしています。

| 機能                                                  | シェアードナッシング | シェアードデータ |
| ---------------------------------------------------- | -------------------- | ---------------- |
| 条件付き更新                                          | v2.5+                | v3.1+            |
| 部分更新 列モード                                     | v3.1                 | サポート予定     |
| 部分更新 行モード                                     | v2.3+                | v3.1+            |
| 条件付き更新との部分更新                              | v3.1+                | v3.4.1+          |

| 主キーテーブルの豊富な UPDATE と DELETE 構文           | v3.0+                | v3.1+            |
