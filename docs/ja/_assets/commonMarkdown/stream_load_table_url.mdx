```Plain
http://<fe_host>:<fe_http_port>/api/<database_name>/<table_name>/_stream_load
```

以下の表は、URL内のパラメーターについて説明しています。

| パラメーター     | 必須 | 説明                                                  |
| ------------- | -------- | ------------------------------------------------------------ |
| fe_host       | はい      | クラスター内のFEノードのIPアドレス。<br />**注**<br />ロードジョブを特定のBEまたはCNノードに送信する場合、そのBEまたはCNノードのIPアドレスを入力する必要があります。 |`8030`。<br />**注**<br />ロードジョブを特定のBEまたはCNノードに送信する場合、そのBEまたはCNノードのHTTPポート番号を入力する必要があります。デフォルトのポート番号は`8030`。 |
| database_name | はい      | テーブルが属するデータベースの名前。         |
| table_name    | はい      | テーブルの名前。                                       |

:::note
[SHOW FRONTENDS](../../sql-reference/sql-statements/cluster-management/nodes_processes/SHOW_FRONTENDS.md) を使用して、FE ノードの IP アドレスと HTTP ポートを確認できます。
:::
