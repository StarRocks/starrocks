# CREATE ROLE

## Description

This statement allows users to create a role.

## Syntax

```sql
CREATE ROLE <role_name>
```

It creates a role with no permission which could be added to the role though "GRANT" command.  

## Examples

 Create a role.

  ```sql
  CREATE ROLE role1;
  ```
