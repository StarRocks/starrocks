# sqrt, dsqrt

## Description

Returns the square root of `x`.

## Syntax

```Haskell
SQRT(x);
```

## Parameters

`x`: supports the DOUBLE data type.

## Return value

Returns a value of the DOUBLE data type.

## Examples

```Plain
mysql> select sqrt(3.14);
+-------------------+
| sqrt(3.14)        |
+-------------------+
| 1.772004514666935 |
+-------------------+
1 row in set (0.01 sec)
```
