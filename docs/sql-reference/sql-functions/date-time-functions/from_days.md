# from_days

## description

### Syntax

```Haskell
DATE FROM_DAYS(INT N)
```

This function returns a date from 0000-01-01.

## example

```Plain Text
MySQL > select from_days(730669);
+-------------------+
| from_days(730669) |
+-------------------+
| 2000-07-03        |
+-------------------+
```

## keyword

FROM_DAYS,FROM,DAYS
