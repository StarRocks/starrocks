# current_timestamp

## Description

Obtains the current date and returns a value if the DATETIME type.

## Syntax

```Haskell
DATETIME CURRENT_TIMESTAMP()
```

## Examples

```Plain Text
MySQL > select current_timestamp();
+---------------------+
| current_timestamp() |
+---------------------+
| 2019-05-27 15:59:33 |
+---------------------+
```

## keyword

CURRENT_TIMESTAMP,CURRENT,TIMESTAMP
