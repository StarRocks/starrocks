# curtime,current_time

## Description

Obtains the current time and returns a vlue of the TIME type.

## Syntax

```sql
TIME CURTIME()`
```

## Examples

```Plain Text
MySQL > select current_time();
+----------------+
| current_time() |
+----------------+
| 15:25:47       |
+----------------+
```

## keyword

CURTIME,CURRENT_TIME
