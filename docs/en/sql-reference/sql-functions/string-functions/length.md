# length



This function returns the length of a string (in bytes).

## Syntax

```Haskell
INT length(VARCHAR str)
```

## Examples

```Plain Text
MySQL > select length("abc");
+---------------+
| length('abc') |
+---------------+
|             3 |
+---------------+
```

## keyword

LENGTH
