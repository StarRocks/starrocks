# space

## Description

Returns a string of the specified number of spaces.

## Syntax

```Haskell
space(x);
```

## Parameters

`x`: the number of spaces to return. The supported data type is INT.

## Return value

Returns a value of the VARCHAR type.

## Examples

```Plain Text
mysql> select space(6);
+----------+
| space(6) |
+----------+
|          |
+----------+
1 row in set (0.00 sec)
```
