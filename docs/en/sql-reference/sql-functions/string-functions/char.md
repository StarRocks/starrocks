CHAR() returns the character value of the given integer value according to the ASCII table.

## Syntax

```Haskell
char(n)
```

## Parameters

- `n`: integer value

## Return value

Returns a VARCHAR value.

## Examples

```Plain Text
> select char(77);
+----------+
| char(77) |
+----------+
| M        |

```
