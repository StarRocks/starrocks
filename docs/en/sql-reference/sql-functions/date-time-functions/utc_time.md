# utc_time

## Description

Returns the current UTC time as a value in 'HH:MM:SS'.

## Syntax

```Haskell
TIME UTC_TIME()
```

## Examples

```Plain Text
MySQL > select utc_time();
+----------------+
| utc_time()     |
+----------------+
| 12:31:18       |
+----------------+
```


## keyword

UTC_TIME,UTC,TIME
