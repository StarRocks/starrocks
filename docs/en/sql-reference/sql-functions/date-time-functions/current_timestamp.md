# current_timestamp

## description

### Syntax

```Haskell
DATETIME CURRENT_TIMESTAMP()
```

Obtain the current date and return in Datetime type.

## example

```Plain Text
MySQL > select current_timestamp();
+---------------------+
| current_timestamp() |
+---------------------+
| 2019-05-27 15:59:33 |
+---------------------+
```

## keyword

CURRENT_TIMESTAMP,CURRENT,TIMESTAMP
