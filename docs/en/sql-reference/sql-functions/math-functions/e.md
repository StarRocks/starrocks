# e

## Description

Returns the constant e (the base of natural logarithms).

## Syntax

```SQL
E();
```

## Parameters

This function takes no parameters.

## Return value

Returns the constant e as an DOUBLE.

## Examples

```Plaintext
mysql> select e();
+-------------------+
| e()               |
+-------------------+
| 2.718281828459045 |
+-------------------+
1 row in set (0.01 sec)
```
