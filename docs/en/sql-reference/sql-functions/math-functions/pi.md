# pi

## Description

Returns the value of Pi.

## Syntax

```Plain
pi()
```

## Parameter

N/A

## Return value

Returns the value of Pi as a DOUBLE type value.

## Examples

```Plain
mysql> select pi();
+-------------------+
| pi()              |
+-------------------+
| 3.141592653589793 |
+-------------------+
1 row in set (0.00 sec)
```
