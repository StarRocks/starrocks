# sin

## Description

Returns the sine of `x`, where `x` is in radians.

## Syntax

```Haskell
SIN(x);
```

## Parameters

`x`: supports the DOUBLE data type.

## Return value

Returns a value of the DOUBLE data type.

## Examples

```Plain
mysql> select sin(3.14);
+-----------------------+
| sin(3.14)             |
+-----------------------+
| 0.0015926529164868282 |
+-----------------------+
1 row in set (0.21 sec)
```
