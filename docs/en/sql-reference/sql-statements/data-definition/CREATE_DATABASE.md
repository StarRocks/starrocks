# CREATE DATABASE

## Description

This statement is used to create databases.

## Syntax

```sql
CREATE DATABASE [IF NOT EXISTS] <db_name>
```

## Examples

1. Create database `db_test`.

    ```sql
    CREATE DATABASE db_test;
    ```
