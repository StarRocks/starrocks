StarRocks provides a self-developed connector named StarRocks Connector for Apache Flink® (hereinafter referred to as Flink connector) to help you load data into the target table by using Flink. The principle is to accumulate multiple data records into one request and then load all the data at a time into StarRocks through [STREAM LOAD](../../sql-reference/sql-statements/loading_unloading/STREAM_LOAD.md).
