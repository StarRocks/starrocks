# CREATE DATABASE

## 功能

该语句用于创建数据库 (database)。

## 语法

```sql
CREATE DATABASE [IF NOT EXISTS] db_name
```

## 示例

1. 新建数据库 db_test。

    ```sql
    CREATE DATABASE db_test;
    ```
