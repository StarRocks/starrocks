# uuid

## 功能

生成一个 32 位的 uuid 值

## 语法

```Haskell
uuid();
```

## 参数说明

无

## 返回值说明

返回值的数据类型为 VARCHAR

## 示例

```Plain Text
mysql> select uuid();
+--------------------------------------+
| uuid()                               |
+--------------------------------------+
| 74a2ed19-9d21-4a99-a67b-aa5545f26454 |
+--------------------------------------+
1 row in set (0.01 sec)
```
