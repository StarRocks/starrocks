# to_days

## 功能

返回指定日期距离`0000-01-01`的天数。

参数必须为 DATE或DATETIME类型。

## 语法

```Haskell
INT TO_DAYS(DATETIME date)
```

## 示例

```Plain Text
select to_days('2007-10-07');
+-----------------------+
| to_days('2007-10-07') |
+-----------------------+
|                733321 |
+-----------------------+
```
