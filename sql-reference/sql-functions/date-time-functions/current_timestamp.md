# current_timestamp

## description

### Syntax

```Haskell
DATETIME CURRENT_TIMESTAMP()
```

获得当前的时间，以Datetime类型返回

## example

```Plain Text
MySQL > select current_timestamp();
+---------------------+
| current_timestamp() |
+---------------------+
| 2019-05-27 15:59:33 |
+---------------------+
```
