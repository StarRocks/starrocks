# ln, dlog1

## 功能

返回参数 `x` 的自然对数，以 e 为底。

## 语法

```Haskell
LN(x);
```

## 参数说明

`x`: 支持的数据类型为 DOUBLE。

## 返回值说明

返回值的数据类型为 DOUBLE。

## 示例

```Plain Text
mysql> select ln(2);
+--------------------+
| ln(2)              |
+--------------------+
| 0.6931471805599453 |
+--------------------+
1 row in set (0.01 sec)
```
