# tan

## Description

Returns the tangent of `x`, where `x` is in radians.

## Syntax

```Haskell
TAN(x);
```

## Parameters

`x`: supports the DOUBLE data type.

## Return value

Returns a value of the DOUBLE data type.

## Examples

```Plain
mysql> select tan(3.14);
+-----------------------+
| tan(3.14)             |
+-----------------------+
| -0.001592654936407223 |
+-----------------------+
1 row in set (0.12 sec)
```
